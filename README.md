# SQL & Excel Sales Analysis Projects
## Project Overview
This repository contains data analysis projects developed using SQL and Microsoft Excel as part of the “Data Analysis | Real Projects” certification program. The project's focus on sales reporting, customer purchasing behavior, KPI tracking, credit limit analysis, and business trend analysis using structured datasets.
The analysis demonstrates practical skills in SQL querying, Excel reporting, pivot table analysis, dashboard visualization, and business intelligence reporting.
# Projects Included
* Sales Overview (2004)
* Customer Sales Trends
* Credit Limit Analysis
* Commented SQL Queries & Data Analysis
# Sales Overview (2004)
## Objectives
* Analyze overall sales performance for the year 2004
* Evaluate regional and product-line sales distribution
* Identify business trends across office countries
* Visualize sales metrics and customer performance
## Analysis Features
* Sales value analysis by office country
* Product line sales comparison
* Regional sales distribution reporting
* Customer country performance tracking
* Pivot table summaries and chart visualizations
## Key Insights
* Identified top-performing office countries based on sales value
* Compared product-line performance across different regions
* Evaluated customer sales contribution by country
* Visualized regional sales distribution trends for business reporting
